# React Tic tac toe

## Getting started

```sh

git checkout main

git pull origin main
```

Create a new brach.

```sh

git checkout -b ex7_tic-tac-toe
```

Init your project.

```sh

npm init -y
```


## Instructions

Follow [this tutorial](https://react.dev/learn/tutorial-tic-tac-toe) and create your tic-tac-toe app using typescript.
