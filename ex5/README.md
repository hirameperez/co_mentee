# Typescript

## Getting started

```sh

git checkout main

git pull origin main
```

Create a new brach.

```sh

git checkout -b ex5_server
```

Init your project.

```sh

npm init -y
```


## Instructions

Based on the exercise number 1, set up your project in order to use Typescript.


You have to create a PR in this repo when you finish.


AC:

- Routes should be located in the `/src/routes` folder.
- Source code should be in the `/src` folder.


References:

https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html
